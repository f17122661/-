# STM32F103 bootloader 简单编写 for IAR

## 1. bootloader 编写

- 这里需要特别注意 在跳转PC指针前需要关闭全局中断，所以在跳转后的APP 程序中需要先打开全局中断 避免定时器调度不开

``` C
/* rct6mcu.h
*FLASH :        0x08000000 --- 0x08040000 Total Size: 256K
*RAM :          0x20000000 --- 0x2000C000 Total Size: 48K


*Bootloader:    0x08000000 --- 0x08004FFF Total Size: 20K
*APP1:          0x08005000 --- 0x0801DFFF Total Size: 100K
*APP2:          0x0801E000 --- 0x08036FFF Total Size: 100K
*OPT:           0x08037000 --- 0x0803FFFF Total Size: 36K
*/
typedef void ( *pFunction )( void );    //定义跳转函数指针类型
/*跳转到应用程序处理函数*/
void JumpToApplication( uint32_t applocationAddr )
{
    uint32_t StackAddr;           //应用程序栈地址
    uint32_t ResetVector;         //应用程序中断向量表的地址
    pFunction JumpToApp;          //定义跳转函数指针
    __set_PRIMASK( 1 );  //关闭全局中断
    StackAddr = *( __IO uint32_t* )applocationAddr;            //0x08004000;
    ResetVector = *( __IO uint32_t* )( applocationAddr + 4 );  //0x08004004;
    if( ( StackAddr & 0x2FFC0000 ) == 0x20000000 ) //检查栈顶地址是否合法.
    {
        __set_MSP( StackAddr );                     //初始化应用程序栈指针
        JumpToApp = ( pFunction )ResetVector;
        JumpToApp();
    }
}
```



## 2. bootloader 工程文件设置

1.  target -> option->linker -> Vector Table 修改.cif 文件 设置中断向量表地址 默认是在 BASE+4
2.  target -> option->linker -> Memory Regions 修改 ROM 区域 这个区域就是 APP 的分配区域
3.  以上两个步骤在编写APP工程的时候也需要放进去

## 3. APP 工程文件编写

*** 这段程序放在` systemInit() ` 函数执行完后面 ***

``` c
SCB->VTOR = APP1_ADD; //给PC指针赋值跳转到指定位置
__set_PRIMASK(0);//开启全局中断
```



## 4. 编译好的bootloader 加载到现有APP 工程文件中一并调试下载

*** 将编译生成好的 BIN 文件转换成 16进制数组 存放到 FLASH 的 0x08000000这个初始引导地址上***

*** 这里特别注意 icf 文件修改主要添加编译环境变量 提前声明 const 变量 放在哪个地址上***

` place at address mem:0x08000000 { readonly section .bootbin };`  这个`.bootbin ` 在程序生命的时候 用 `__root const unsigned char STM32F10x_BootLoader[] @".bootbin"= {};`

``` cif
/*###ICF### Section handled by ICF editor, don't touch! ****/
/*-Editor annotation file-*/
/* IcfEditorFile="$TOOLKIT_DIR$\config\ide\IcfEditor\cortex_v1_0.xml" */
/*-Specials-*/
define symbol __ICFEDIT_intvec_start__ = 0x08005000;
/*-Memory Regions-*/
define symbol __ICFEDIT_region_ROM_start__ = 0x08005000;
define symbol __ICFEDIT_region_ROM_end__   = 0x0801DFFF;
define symbol __ICFEDIT_region_RAM_start__ = 0x20000000;
define symbol __ICFEDIT_region_RAM_end__   = 0x2000BFFF;
/*-Sizes-*/
define symbol __ICFEDIT_size_cstack__ = 0x2000;
define symbol __ICFEDIT_size_heap__   = 0x2000;
/**** End of ICF editor section. ###ICF###*/

define memory mem with size = 4G;
define region ROM_region   = mem:[from __ICFEDIT_region_ROM_start__   to __ICFEDIT_region_ROM_end__];
define region RAM_region   = mem:[from __ICFEDIT_region_RAM_start__   to __ICFEDIT_region_RAM_end__];

define block CSTACK    with alignment = 8, size = __ICFEDIT_size_cstack__   { };
define block HEAP      with alignment = 8, size = __ICFEDIT_size_heap__     { };

initialize by copy { readwrite };
do not initialize  { section .noinit };


place at address mem:0x08000000 { readonly section .bootbin };
place at address mem:__ICFEDIT_intvec_start__ { readonly section .intvec };

place in ROM_region   { readonly };
place in RAM_region   { readwrite,
                        block CSTACK, block HEAP };
```
